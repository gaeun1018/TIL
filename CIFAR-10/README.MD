# CIFAR-10 Image Classifier

## 📌 프로젝트 소개
CIFAR-10 이미지 분류를 위한 CNN 기반 모델을 구현하고, 다양한 실험을 통해 성능을 개선해 보았습니다.


## 🧾 사용한 기술
- PyTorch
- CNN (Convolutional Neural Network)
- Data Augmentation
- Batch Normalization, Dropout

## 📊 모델 성능
| Model | Accuracy |
|-------|----------|
| Baseline CNN | 72% |
| + BatchNorm | 78% |
| + Augmentation | 81% |

## 📎 주요 코드
- `model.py`: 모델 정의
- `train.py`: 학습 루프
- `eval.py`: 테스트 및 정확도 확인
