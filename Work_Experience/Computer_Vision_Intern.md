# Work Experience – Computer Vision Intern

## 📌 개요
이미지 인식 기반 솔루션을 개발하는 기업에서 Computer Vision 인턴으로 근무하며   
**이미지 전처리, 데이터 수집, 모델 실험 및 성능 개선 업무**를 수행했습니다.

근무 형태: 인턴
주요 분야: Image Processing, Computer Vision, Deep Learning

---

## 🛠  주요 업무 및 기여

### 1️⃣ 이미지 데이터 전처리 및 정제
- 학습 데이터 품질 향상을 위한 **이미지 전처리 파이프라인 구성**
- OpenCV 및 Python 기반 이미지 처리 수행
  - Resize, Crop, Noise 제거
  - 밝기 및 대비 조정
- 얼굴 인식 모델 입력 조건에 맞게 **데이터 구조 및 포맷 정리**
- 실제 서비스 환경에서 수집된 이미지 특성을 고려한 데이터 정제

📌 데이터 품질 개선을 통해 모델 학습 안정성 및 추론 결과의 일관성 향상

---

### 2️⃣ 얼굴 인식 모델 구조 분석 및 설정 수정
- 기존에 구축된 **Face Recognition 모델 구조 분석**
- 입력 이미지와 출력 결과 간의 관계 파악
  - 입력 이미지 크기 및 채널 구성
  - 출력 형태 (embedding 벡터 또는 클래스 분류 결과)
- 모델 추론 과정에서 사용되는 전처리 및 후처리 방식 확인
- 프로젝트 요구사항에 맞게 **모델 설정값 수정**
  - 입력 차원 및 데이터 포맷 조정
  - 출력 구조(embedding 차원 또는 클래스 수) 변경
- 설정 변경 후 추론 결과 검증 및 오류 분석

📌 기존 모델을 블랙박스로 사용하지 않고,  
   데이터–모델–출력 구조를 이해하며 실제 환경에 맞게 조정한 경험

---

### 3️⃣ 이미지 인식 모델 실험 및 성능 개선
- CNN 기반 이미지 인식 모델 구조 분석
- 기존 모델의 성능 병목 구간 파악
- 하이퍼파라미터 튜닝 실험 수행
  - Learning rate
  - Batch size
  - Optimizer 변경
- 실험 결과 비교를 통한 성능 변화 분석

📌 모델 구조 및 학습 전략 변경이 성능에 미치는 영향을 직접 실험하고 비교 분석

---

### 4️⃣ 데이터 수집 및 자동화 (크롤링)
- Python 기반 이미지 크롤링 스크립트 작성
- 대규모 이미지 데이터 수집 자동화
- 수집 데이터의 중복 제거 및 라벨 정리
- 모델 학습에 적합한 형태로 데이터셋 구성

📌 학습 데이터 확보 과정을 자동화하여 작업 효율성 향상

---

### 5️⃣ 실험 결과 정리 및 공유
- 실험 로그 및 결과를 **Markdown 기반 문서로 정리**
- 팀 내부 공유를 위한 문서화 수행
- 모델 성능 비교 결과 및 개선 방향 제안

📌 실험 결과를 단순 수치가 아닌  
   원인–결과 중심의 해석 가능한 형태로 정리
